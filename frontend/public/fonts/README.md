# Local web fonts

Place these font files in this folder to serve fonts locally:

- `syne-400.woff2`
- `syne-500.woff2`
- `syne-600.woff2`
- `syne-700.woff2`
- `syne-800.woff2`
- `ibm-plex-mono-400.woff2`
- `ibm-plex-mono-500.woff2`

They are referenced from `frontend/assets/css/main.css` using `/fonts/...` URLs.
